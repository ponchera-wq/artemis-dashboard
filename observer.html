🛰️ •• ABOUT THE OBSERVER TOOL [–]
This application is a 'vibe code' project translating the astrophotography calculations and logic developed by Shawn Gano into a live web dashboard. For the original Python scripts and complete Artemis II imaging guide, visit gano.name/shawn/artemis2 or watch the overview video.
PRIVACY NOTE: ALL CALCULATIONS ARE PERFORMED LOCALLY. NO COORDINATE, SENSOR, OR FOCAL DATA IS RECORDED, SAVED, OR TRANSMITTED TO ANY EXTERNAL SERVER.
